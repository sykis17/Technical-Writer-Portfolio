    This page defines the specialized terminology and communication standards used throughout this portfolio and within flight operations environments.

    1. Project & Workflow Terms

    Term Definition
    WIP Work In Progress. Indicates a section currently under development, research, or drafting phase.
    Docs-as-Code A philosophy where documentation is written using the same tools as software (Git, Markdown, VS Code).
    LEP List of Effective Pages. A master list used to verify the current revision status of every page in a manual.
    SOP Standard Operating Procedure. A set of step-by-step instructions compiled to help workers carry out complex routine operations.

    3. Operational Abbreviations

    Abbreviation Full Term Context
    FCOM Flight Crew Operating Manual Includes for aircraft-specific systems and procedures.
    QRH Quick Reference Handbook Used for non-normal or emergency checklist execution.
    VNAV Vertical Navigation An FMC function that calculates a vertical profile for climb, cruise, and descent.
    CRM Crew Resource Management The effective use of all available resources to ensure flight safety.
    AFDS Autopilot Flight Director System The integrated system controlling both the flight director and autopilot.
    CDU Control Display Unit The physical interface (keyboard/screen) used to input data into the FMC.
    EFIS Electronic Flight Instrument System The digital display system (PFD/ND) used to present flight data.
    FMC Flight Management Computer The "brain" of the aircraft that calculates performance, navigation, and vertical paths.
    MCP Mode Control Panel The interface located on the glare shield used to select autopilot modes (Heading, Altitude, Speed).
    PFD Primary Flight Display The main screen providing attitude, airspeed, altitude, and vertical speed.
    ND Navigation Display The map display showing the route, weather radar, and TCAS information.
    TCAS Traffic Collision Avoidance System An airborne system designed to increase cockpit awareness of nearby aircraft.
    WCN Warning, Caution, Note The standardized hierarchy of safety information used in all Boeing manuals.
