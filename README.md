# B737 Technical Documentation Portfolio

This repository hosts a flight operations documentation suite built using the **Docs-as-Code** workflow. 

## Project Objective
To demonstrate the ability to manage safety-critical aviation manuals using software development tools, ensuring high accuracy and rapid deployment capabilities.

## Tech Stack
* **Engine:** MkDocs with Material Theme
* **Language:** Markdown & HTML5
* **Version Control:** Git

## Local Deployment
To view this portfolio locally:
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git`
2. Install requirements:
   `pip install mkdocs-material`
3. Start the server:
   `mkdocs serve`
